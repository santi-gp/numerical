Conjunto de Mandelbrot

Programa

El conjunto de Mandelbrot M esta compuesto por todos los valores c para el cual el lleno del conjunto de Julia Kc es acotado, es decir,

M = {c ∈ C / Qcn(0) ↛ ∞ }

donde c es una constante compleja llamada parámetro.

La Dicotomía Fundamental indica que existen 2 tipos básicos de conjuntos Llenos de Julia para fc, aquellos que son conexos y otros compuestos por infinitos componentes disjuntos. Además es la órbita de 0 la que determina estos 2 casos.

Como antes le animamos a experimentar viendo diferentes regiones de M. Usted verá rápidamente que el conjunto de Mandelbrot es uno de los objetos más fantásticos en la matemáticas. Intentemos entender que significa esta región.

Supongamos que Qc tiene un punto periódico atractivo de algún periodo. Entonces la cuenca entera de atracción de este punto se encuentra en Kc, y así el lleno del conjunto de Julia no es un conjunto de Cantor. Por nuestra discusión anterior, esto significa que c debería estar en el conjunto de Mandelbrot. La pregunta es ¿Dónde?.

Para responder esto, primero se determina el conjunto de valores de c para el cual Qc tiene un punto fijo atractivo o un punto fijo neutral. Denotemos este conjunto por C1 y supongamos que zc representa el punto fijo correspondiente a Qc cuando c ∈ C1. Entonces zc debería satisfacer el siguiente par de ecuaciones:

Qc(z) = z2 + c = z
|Q'c(z)| = |z| ≤ 1.

La segunda ecuación afirma que zc en el interior del disco acotado por el círculo de radio 1/2. En este círculo |Q'c(z)|=1, zc es un punto fijo neutral.
