Factorización LU

Programa

 

Lo que vamos a hacer es mostrar que podemos factorizar la matriz A en el producto de un matriz triangular inferior y una matriz triangular superior:

A = LU.

Esto nos permite resolver los sistemas lineales mediante la resolución de dos sistemas triangulares:

Ax= b ⇒ Ux = y, donde Ly = b.

Por lo tanto, primero resolvemos Ly=b y luego Ux=y para obtener la solución:

Ux = y, Ly = b ⇒ Ux = L-1b
⇔ LUx = b ⇔ Ax = b.

Esto demuestra que podemos resolver sistemas lineales mediante

  1. Calcular la factorización LU de A;
  2. A continuación, la solución de sistemas triangulares correspondientes inferior y superior.
