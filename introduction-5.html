Solución de sistemas lineales

En el apartado I consideramos la solución única de una ecuación de la forma f(x) = 0 donde f es una función de valor real definida y continua en un intervalo cerrado de la recta real. El ejemplo más simple de este tipo es la ecuación lineal ax = b donde a y b son números reales, con a ≠ 0, cuya solución es x = a-1b

Por supuesto, podríamos haber expresado la solución como x = b/a como en el apartado I, pero, como se verá a continuación, escribir x = a-1b es mucho más ilustrativo en el contexto actual. En este apartado consideraremos una generalización diferente de este problema elemental:

Sea A una matriz n × n con aij como su entrada en la fila i y la columna j y b un vector de columna dado de tamaño n con j-ésima entrada bj; Encuentre un vector columna x de tamaño n tal que Ax = b.

Denotando por xi la i-ésima entrada del vector x, también podemos escribir Ax = b en la siguiente forma expandida:

a11x1 + a12x2 + … + a1nxn = b1
a21x1 + a22x2 + … + a2nxn = b2
  ⋮  
an1x1 + an2x2 + … +annxn = bn

Recordemos que para asegurar que para los números reales a y b la ecuación lineal simple ax = b tiene una solución única, debemos suponer que a ≠ 0. En el caso del sistema simultáneo anterior de n ecuaciones lineales con n incógnitas tendremos que hacer una suposición análoga en la matriz A.

Para ello, introducimos la siguiente definición.

Se dice que una matriz A ∈ Rn×n es no singular cuando su determinante det(A) es distinto de cero.

La matriz inversa A-1 de una matriz no singular A ∈ Rn×n se define como el elemento de Rn×n tal que A-1A = AA-1 = I, donde I es la matriz de identidad n×n

Habiendo encontrado una fórmula explícita para la matriz A-1, ahora multiplicamos ambos lados de la ecuación Ax = b a la izquierda por A-1 para deducir que A-1(Ax)=A-1b; Finalmente, dado que A-1(Ax)=(A-1A)x=Ix=x, se sigue que

x=A-1b.
