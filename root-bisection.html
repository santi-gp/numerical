Método Bisección

Programa

 

La bisección es una idea maravillosamente sencilla que se basa en poco más que la continuidad de la función f.

Supongamos que sabemos que f(a)f(b) < 0 en el intervalo [a,b]. Esto significa que f es negativa en un punto y positiva en el otro. El teorema del valor intermedio para funciones continuas implica que existe un número p en (a,b) con f(p) = 0. Aunque el procedimiento funciona cuando hay más de una raíz en el intervalo (a,b), asumimos por simplicidad, que la raíz en este intervalo es única.

El método pide una reducción a la mitad repetida de subintervalos de [a,b] y, a cada paso, localizar el intervalo que contiene p.

Para empezar, sea a1 = a y b1 = b, y sea p1 el punto medio de [a, b] ; es decir,

p1 =
a1 + a2
2

Si f(p1) = 0, entonces p = p1, y hemos terminado.

Si f(p1) ≠ 0, entonces f(p1) tiene el mismo signo que f(a1) o f(b1).

- Si f(p1) y f(a1) tienen el mismo signo, p∈(p1,b1). Hacer a2 = p1 y b2 = b1.

- Si f(p1) y f(a1) tienen signos opuestos, p∈(a1,p1). Hacer a2 = a1 y b2 = p1.

A continuación, vuelva a aplicar el proceso para el intervalo [a2,b2].
