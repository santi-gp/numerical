Método de Euler

Programa

 

El método más sencillo de todos los métodos numéricos para ecuaciones diferenciales ordinarias es el método de Euler. Aunque este método se utiliza raramente, es relativamente fácil de analizar.

Supongamos que queremos aproximar la solución del problema de valor inicial

dy
dt
= f(t,y),  y(a) = y0.

en algún intervalo [a, b].

Para comenzar, primero dividimos el intervalo [a, b] en n subintervalos iguales y definamos los puntos de malla

ti = a + ih,   i = 0,...,n.

La distancia común entre los puntos h=(b-a)/n se denomina tamaño de paso.

Para derivar el método de Euler, consideramos la expansión de Taylor de y sobre ti para cada i=0,1,...,n-1. Supongamos que y(t) es dos veces continuamente diferenciable en [a, b], entonces

y(ti+1)=y(ti) + hy'(ti) +
h2
2
y''(ξi),

para algún número ξi entre (ti, ti+1). Debido a que y(t) satisface la ecuación diferencial y'(t)=f(t,y(t)) y eliminando el término de error, obtenemos la fórmula

y(ti+1) ≈ y(ti) + hf(ti,y(ti)).

Si denotamos yi = y(ti), entonces

yi+1 ≈ yi + hf(ti,yi).

Esta última ecuación se conoce como método de Euler.
